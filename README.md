# Canvas Pocket

上海交通大学 Canvas（[oc.sjtu.edu.cn](https://oc.sjtu.edu.cn/)）课程课堂录像下载工具。

## 下载

前往 [Releases](https://github.com/FengYuchen1314/canvas-downloader/releases) 下载对应系统的安装包：

| 系统 | 格式 |
|------|------|
| Windows | `.exe` |
| macOS | `.dmg` |
| Linux | `.deb` / `.rpm` |

下载后安装或解压，打开应用即可使用。

## 项目来源

本项目是对 [sjtu-canvas-video-download](https://github.com/prcwcy/sjtu-canvas-video-download) 的二次开发，在保留其核心下载思路的基础上，重新实现了登录、界面与下载管理等功能。

## 功能

- 交我办扫码登录 Canvas
- 浏览全部课程（进行中 / 已结束）
- 按讲次、分轨（教师 / PPT）批量下载
- 下载任务管理（暂停 / 继续 / 取消）
- 断点续传与并发下载

## 限制说明

**不支持旁听课程下载。** 仅展示并下载审核状态为「开放」的讲次录像。旁听或未开放审核的讲次不会出现在列表中，也无法通过本工具下载。
